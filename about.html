About the Purdue University Dynamic Parking Map

Purdue University Dynamic Parking Map is a personal project developed to explore and learn more about geospatial data, interactive web mapping, and GIS tools like Leaflet.js, GeoJSON, and ArcGIS Feature Layers.

This interactive web application displays real-time data from Purdue University's ArcGIS Online portal, offering users a dynamic view of campus parking lots and buildings. While the data is sourced from Purdue, this application is not affiliated with or maintained by the university.

🚗 Project Overview

The goal of this project is to gain hands-on experience working with:

  • Live geospatial datasets
  • Spatial queries and filters
  • Web mapping tools and libraries

Users can:

  • View real-time parking lot and campus building data
  • Filter parking types by category
  • Access directions and explore campus layout interactively

📍 Data Sources

All geospatial data is publicly available and accessed directly from Purdue University’s ArcGIS Online portal:

  • Purdue Campus Buildings (Feature Layer)
  • Parking 2020 – Lots & Garages (Feature Layer)
These datasets are integrated live using Esri’s FeatureService endpoints.

🔧 Technologies Used

  • Leaflet.js – For creating interactive maps
  • Esri Leaflet – For integrating ArcGIS Feature Layers
  • HTML / CSS / JavaScript – Core frontend development
  • GeoJSON & ArcGIS REST Services – For handling spatial data
